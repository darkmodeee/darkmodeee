# DarkMode ~ Web App

### PROJECT SETUP
```
npm install
```

### COMPILES AND HOT-RELOADS FOR DEVELOPMENT
```
npm run serve
```

### COMPILES AND MINIFIES FOR PRODUCTION
```
npm run build
```

### RUN YOUR UNIT TESTS
```
npm run test:unit
```

### LINTS AND FIXES FILES
```
npm run lint
```
